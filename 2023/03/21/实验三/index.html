1.注册github账号

账号地址：（Liuzhao2022 (github.com)）

创建个人仓库
  • 仓库名为 Liuzhao2022.github.io
  • 仓库地址为：https://github.com/Liuzhao2022/Liuzhao2022.github.io.git

2.下载Node.js和Git

  • 下载地址：（下载 | Node.js 中文网 (nodejs.cn)）

3.安装配置hexo

  • 在磁盘中创建一个用来存放Github本地仓库文件的目录(可自定义目录存放，顾名思义blog==博客)
  • 选中创建的目录(文件夹)右键选择使用Gti Bash Here打开Git命令窗口，输入命令：npm install -g cnpm –registry=https://registry.npm.taobao.org
  • 输入命令：cnpm install -g hexo-cli，正式安装hexo
  • 初始化Hexo

此时文件夹下出现了很多文件

  • 启动hexo
  • 打开浏览器输入localhost:4000/，就会出现一篇自带Hello World的博客

4.设置ssh

  • 输入命令：cd ~/.ssh进入.ssh文件
    输入命令：ssh-keygen -t rsa -C ‘注册时的邮箱地址’
  • 使用记事本打开id_rsa.put文件并复制生成的key
  • 在已登陆的Github主页点击右侧头像 》 Settings设置
  • 再点击SSH and GPG keys 》 New SSH key
  • 将刚刚在.ssh目录下所复制id_rsa.put文件中的信息复制进key(注意空格)，在给它取个名字(随意)
  • 这里需要在本地进行验证一次,输入命令：ssh -T git@github.com 并且输入yes之后，行末尾会显示你的用户名
  • 绑定成功并且邮箱收到邮件
  • 接着在本地绑定与Github的用户名和邮箱
    输入命令：git config –global user.name “注册时用户名”
    输入命令：git config –global user.email “注册时邮箱”

5.上传测试博客

5.1用记事本打开并修改本地仓库目录下_config.yml文件

在文件的末尾修改(repository属性的地址就是之前创建仓库后所保存的http地址)

5.2此时需要安装一个上传工具

输入命令：cnpm install hexo-deployer-git

5.3可以新建一篇测试文章(不新建的话也会有一篇自带的Hello World文章)

​ 输入命令：hexo new “文章名称”

5.4新建文章之后需要生成一遍文件

输入命令：hexo g

5.5新建文章后可在本地先预览一遍

输入命令：hexo s浏览器输入：localhost:4000(ctrl + c 关闭)

5.6最后直接部署到Github就可以啦

输入命令：hexo d

5.7 github仓库上多出了一些文件

5.8此时再浏览器输入：Liuzhao2022.github.io(Github仓库名)就可以访问博客啦
